tclBlend141Binary.zip:

This is the Tcl Blend 1.4.1 binary release.
The contents of this zip file must be installed
into C:/ so that path of this README.txt file
is C:/TclBlend/README.txt.

The following software is included:

JDK 1.4.2.15 Multi-Language Edition
  j2sdk-1_4_2_15-windows-i586-p.exe

Tcl/Tk 8.4.15, Thread 2.6.4, Tcl Blend 1.4.1
  tcl8415-src.zip
  tk8415-src.zip
  thread264.zip
  tclBlend141.zip

The JDK is installed in:

  C:\TclBlend\j2sdk1.4.2_15

Tcl, Tk, Thread, and Tcl Blend are installed in:

  C:\TclBlend

A Tcl Blend enabled Tcl or Tk shell can be run via:

  C:\TclBlend\bin\jtclsh.exe
  C:\TclBlend\bin\jwish.exe

Enter "package require java" after starting jtclsh
or jwish to load the Tcl Blend package.
